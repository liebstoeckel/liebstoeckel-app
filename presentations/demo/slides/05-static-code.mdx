## Static code blocks just work

Fenced Markdown is highlighted at build time — on-brand, zero runtime cost:

```ts
import { CodeMagic } from "@present-it/engine";
import { codeStory } from "@present-it/engine/code" with { type: "macro" };

// tokenized at build, animated at runtime
const story = codeStory([{ code: "const x = 1", lang: "ts" }]);
```
