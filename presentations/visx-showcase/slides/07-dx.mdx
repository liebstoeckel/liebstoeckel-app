# Authoring is just React.

This very slide is **MDX**. The chart slides are **TSX**. Mix them freely.

- Write prose in **Markdown**, drop in any **React** component
- Charts are plain **visx** — `import`, render, done
- `bun run build` → one portable **.html**, zero runtime deps
- Hot reload while you write; press **P** for the presenter view
